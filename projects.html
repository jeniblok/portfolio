Lavender Steel Forge Website

Project Description:

This project is a custom-built website developed for Lavender Steel Forge, a blacksmithing business. The purpose of the site is to showcase products, provide business information, and create a professional online presence for potential customers.

Technologies Used:

  • HTML5
  • CSS3
  • Responsive Web Design
  • Media Integration (Images & Video)

Key Features:

  • Custom layout and navigation structure
  • Responsive design for mobile and desktop users
  • Image gallery showcasing products and craftsmanship
  • Video integration for visual engagement
  • Contact page for customer inquiries

Sample Code:

document.addEventListener("DOMContentLoaded", function () { const form = document.getElementById("contactForm"); const formMessage = document.getElementById("formMessage"); if (form) { form.addEventListener("submit", function (event) { event.preventDefault(); const name = document.getElementById("name").value.trim(); const email = document.getElementById("email").value.trim(); const phone = document.getElementById("phone").value.trim(); const project = document.getElementById("project").value.trim(); if (!name || !email || !project) { if (formMessage) { formMessage.textContent = "Please complete the required fields before submitting."; } return; } const formData = { name, email, phone, project, date: new Date().toLocaleString() }; const submissions = JSON.parse(localStorage.getItem("forgeRequests")) || []; submissions.push(formData); localStorage.setItem("forgeRequests", JSON.stringify(submissions)); if (formMessage) { formMessage.textContent = "Thanks! Your request has been saved on this browser."; } form.reset(); }); } const faqButtons = document.querySelectorAll(".faq-question"); faqButtons.forEach((button) => { button.addEventListener("click", function () { const item = button.closest(".faq-item"); const isOpen = item.classList.contains("open"); item.classList.toggle("open"); button.setAttribute("aria-expanded", String(!isOpen)); }); }); });

This JavaScript code handles form validation, user input processing, and interactive UI behavior for the Lavender Steel Forge website. It prevents form submission when required fields are missing, stores submitted data in the browser using localStorage, and provides user feedback. Additionally, it controls the FAQ section by toggling visibility when questions are clicked, improving user experience.
